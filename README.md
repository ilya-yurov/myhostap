Для запуска приложения необходимо самостоятельно присвоить устройству MAC-адрес.
Сделать это можно следующим образом:
1. ifconfig [wlanX] down
2. ifconfig [wlanX] hw ether D0:0E:A4:81:FC:00
3. ifconfig [wlanX] up
